Batch Processing Steps

Step 1:In this step it Read the values from Excel to JSON for each city. Begin by uploading a JSON file with initial data. Then, select Excel files containing additional information for cities. After processing, download a ZIP file containing JSON files, each json for a specific city.

Step 2: Normalize Data. In this step, we ensure all data is on the same scale to ensure fair comparisons. To begin normalization, click on the "Step 2" button, and then load the folder of JSON files that you just downloaded from Step 1. Once the normalization process is complete, you will find the normalized data for each city.

Step 3: Assign Values to Internal Nodes. In this step, we assign values to internal nodes based on the normalized values of their children. Click the "Step 3" button to start the process. Ensure you have the normalized JSON data from Step 2 ready to load. After processing, you will obtain the assigned values for each internal node and also this is final step where you will have final values for each types of node.
