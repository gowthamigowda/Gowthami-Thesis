Welcome to the urban data management application!

This project's purpose is to provide an interactive tool for visualising urban data indicators. The tool will allow users to compare numerous indicators across cities, assisting decision-makers in developing successful management strategies.

  • Generate data for urban data management: The functionality includes the ability to generate data for urban management, either by creating a hierarchical dataset from scratch or loading existing indicator data. Additionally, the application supports data-saving features to preserve generated insights.
  • Batch Processing: When you click on "Batch Processing," the application performs a series of operations to process the data. First, it loads hierarchies from the input setting, followed by fetching indicator values for all cities and years from the provided Excel file. Then, for each city/year combination, it generates a hierarchy.
  • Next, the application propagates the values through the hierarchy to assign values to internal nodes. This procedure results in a collection of hierarchies filled with values for each city, year, and indicator.
  • Visualization data for urban data management: When you click on "Visualization data for urban data management," the application allows you to compare the data of different cities by selecting them from a dropdown menu. The data is visualized using a sunburst chart, which displays two hierarchies. By clicking on outer layers of the sunburst, you can view the comparison of attached indicator data for selected cities based on normalized values. Additionally, you can view this indicator data in the form of a histogram. Furthermore, whenever you change the year from the dropdown menu, the histogram dynamically updates to display data corresponding to the selected year. From this page, you can also explore one city versus another city, one city versus all other cities' average, and all cities together comparisons.
